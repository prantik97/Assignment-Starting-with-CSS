BEM stands for Block, Element, and Modifier. It's a CSS naming convention for writing cleaner and more readable CSS classes.

Above I've used classname, "card" as a Block. "card--dark" and "card--light" as a Modifier. "card__image","card__body","card__title" as Elements which follows the BEM naming convension.
